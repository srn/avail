# avail [![Build Status][travis-badge]](https://travis-ci.org/srn/avail)

> Check if a domain is available using the CLI. Also outputs whois information.

Using the [Domainr](https://domainr.com/) v2 API.

## CLI

```sh
$ npm install -g avail
```

```sh
$ avail --help

  Example
    avail avail.io

    ✔ avail.io︎
    ✖ avail.com
```
